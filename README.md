# World Cup 2026 Shiny Dashboard

Interactive dashboard built in R Shiny to explore FIFA World Cup data.

## Features
- Filter teams by group and confederation
- Explore stadium capacities and host cities
- Analyze historical World Cup winners
- Interactive charts and tables

## Live App
https://evan-spicer.shinyapps.io/world-cup-clean

## Tech Stack
- R
- Shiny
- ggplot2
- DT
